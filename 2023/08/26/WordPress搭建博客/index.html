WordPress搭建博客

Created2023-08-26|Updated2023-08-27|blog
|Word count:518|Reading time:1min|Post View:

1.准备工作

下载VMware、Xshell、Xftp

让Xshell连接虚拟机、后续操作在Xshell里进行

2.在Terminal内登录服务器并进行相关配置

2.1 在Terminal内先登录到ubuntu用户(腾讯云特有,其他云平台用root用户登录)

ssh ubuntu@124.222.129.254

2.2 公钥获取方法

# Terminal内输入命令
ssh-keygen

# 获取公钥
vim .ssh/id_rsa.pub

2.3 修改ssh相关的设置并配置ssh密钥登陆root用户

# 进入ssh_config文件
vim /etc/ssh/sshd_config

# 修改配置
PermitRootLogin prohibit-password

PubkeyAuthentication yes

PasswordAuthentication no
PermitEmptyPasswords no

# 通过systemctl 重启ssh服务
systemctl restart sshd

# 进入root用户
sudo -i

# 在authorized_keys文件夹里放公钥(可以放多个,用逗号隔开)
vim /.ssh/authorized_keys

# 退出到终端 即可用root用户登录
ssh root@124.222.129.254

# 配置快捷登录 在Terminal里的.ssh/config里配置Host HostName User
vim .ssh/config

# 登录服务器 在.ssh/authorized_keys里配置公钥
# 即可用快捷方式登录

3. 安装宝塔

宝塔官网地址: 宝塔面板下载)

# 在服务器端用ubuntu的脚本进行安装
wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh ed8484bec

宝塔命令

# 启动
/etc/init.d/bt start

# 停止
/etc/init.d/bt stop

bt default

# 修改密码
bt 5

# 打开宝塔面板
/etc/init.d/bt

登录宝塔如果登陆不上去怎么办？

需要放开端口: 防火墙添加一条ALL的规则 添加了之后要注意有些端口不能乱开
