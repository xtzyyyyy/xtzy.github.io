01 电子相册案例

快捷键

~ : 放大项目面板

alt + 滚轮: 调整素材大小

\: 自动调整

v : 选择工具

C: 剃刀工具

ctrl + c: 复制

Ctrl + alt + v: 黏贴属性

新建序列

新建序列的参数:

  • 编辑模式: 自定义
  • 时基: 25帧/秒
  • 帧大小: 1920 水平 1080 垂直
  • 像素长宽比: 方形像素
  • 场: 无场扫描

批量添加转场效果:

先右键 添加为 默认过度 再对素材进行框选 使用 Ctril + D进行添加

交叉溶解 的持续时间一般16-17帧即可。

批量修改效果的持续时间

[编辑]->[首选项]->[时间轴]

相册分层+制作边框:

选中素材 按住 alt往上托,选中上面的一张图片 缩放70%左右

添加[径向阴影],拉到选中图片上,设置参数: 调整图层大小 阴影颜色改成白色 不透明度改成100%

光源参数为： x:1000 y:530 投影距离: 5左右

添加效果:投影

[效果]里[透视]找[投影] 参数:距离(30)和柔合度调整

添加效果:高斯模糊

注: 需要添加到下一层的图片上,以免相片出现模糊,再调整模糊度

批处理

选中已经做好的一个图片,右击复制,然后框选其他的素材,选择[粘贴属性]

添加效果: 立方体旋转

拖到两个素材中间自动应用,

添加光晕效果

[添加]素材,选择不透明度,混合模式改为绿色。修改完之后将素材填满

导出

格式: H264

预设: 高比特率

04 手机 拍摄和剪辑

曝光三要素

  • 快门
  • 光圈
  • ISO

快门

决定感光元件前幕帘的闭合快慢,数字越大,快门越慢 一般1/50 即可(除非拍摄特殊画面,比如车水马龙的流光效果)

快门越慢就越亮

影响:

  • 亮暗
  • 动态模糊

结论:

2倍原则,即视频帧数的两倍

30Fps的视频采用1/60的快门

24Pfs的视频采用近似的1/50的快门

适当动态模糊,模拟人眼视觉

如果再LED光源下拍视频,录像带快门速度设为50的倍数可以繁殖频闪

光圈: 决定相机镜头打开的大小

数字越大 光圈越小

光圈越小就越暗

影响:

亮暗

景深,使背景虚化

结论:

突出主体人物: F2.8大光圈、背景虚化

整体画面清晰:F4.0小光圈、背景清晰

背景虚化F2.8左右,不用虚化F5.0以上

手机的硬件条件有限,一般无法改变光圈大小

ISO：相机感光元件本身的敏感度

数字越大、感光元件对光线越敏感

影响:

亮暗

噪点

结论:

保证正常曝光,ISO越低越好

看环境灯光,避免弱光下的噪点,ISO一般1600以下即可 画面太暗的话,需要灯光辅助,或调大光圈

手机拍摄参数设置

FiLMic软件(正版收费)： 视频也能调整参数

手机辅助设备

  • 手机三脚架

  • 八爪鱼

  • 三脚架

  • 手机稳定器

  • 俯拍支架

手机剪辑软件

  • 巧影
  • 喵影工厂
  • InShot
  • 剪映
