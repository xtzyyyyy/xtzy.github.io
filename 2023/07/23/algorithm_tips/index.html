1.C++如何让输出保存四位小数

#include <iomanip> //引入iomanip头文件
double a=123.4567890;
cout<<setiosflags(ios::fixed)<<setprecision(4)<<a<<endl;  //输出的a就会保留四位小数了。

·

2.如何寻找数组中有多个最大值的最小下标？·

最简单的方法:将原数组复制一遍到另一个数组 将原数组排序 保存最大值 将另一个数组遍历 第一次遇到最大值就输出 break；

3.求最大公约数 和 最小公倍数

int gcd(int a, int b){
	return b > 0 ? gcd(a , b % a) : a;
}

//最大公倍数
int lcm = a * b / GCD(a,b); //注意,这种写法可能错,因为a * b可能因为太大 超出int或者超出longlong 推荐写成int lcm = a / gcd * b

4.洛谷p1420 最大连号 （和最长上升子序列相似）

朴素方法:直接找每次最长的len 一旦遇到不符合要求的 直接将len 置 0 max = len 最后记得要算上 + 1

5.C++字符串中的字母大小写转换

#include <ctype.h>

int main()
{
	string str;
	cin >> str;

	for(int i = 0; i < str.size(); i ++)
		str[i] = toupper(str[i]); //转大写 tolower(str[i]); 转小写
	
	return 0;
}

​ 2.对于去除重合部分，如果没有思路的话,可以考虑一下用染色的方法(0 1来判别) 初始化全0 染过的就置为1 染过的就不能再染了(根据 != 1 判别)
